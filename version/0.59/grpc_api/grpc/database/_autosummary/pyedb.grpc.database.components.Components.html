class pyedb.grpc.database.components.Components(p_edb: Any)#

Manages EDB components and related methods accessible from Edb.components.

Parameters:
p_edbpyedb.grpc.edb.Edb

EDB object.

Examples

>>> from pyedb import Edb
>>> edbapp = Edb("myaedbfolder")
>>> edbapp.components
