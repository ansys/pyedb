





:class:`BatchInfo`
==================


.. py:class:: pyedb.workflows.utilities.siwave_log_parser.BatchInfo

   
   Batch simulation run metadata.



   :Attributes:

       **path** : :class:`python:str`
           Full path to the project file.

       **started** : :obj:`datetime`
           Simulation start timestamp.

       **stopped** : :obj:`datetime`
           Simulation stop timestamp.

       **run_by** : :class:`python:str`
           Username who executed the simulation.

       **temp_dir** : :class:`python:str`
           Temporary directory used during simulation.

       **project_dir** : :class:`python:str`
           Project working directory.

       **status** : :class:`python:str`, :obj:`optional`
           Simulation completion status such as ``'Normal Completion'`` or ``'Aborted'``.
           The default is ``""``.












   .. rubric:: Examples

   >>> from datetime import datetime
   >>> batch = BatchInfo(
   ...     path="C:\project\design.siw",
   ...     started=datetime(2025, 11, 10, 9, 0, 0),
   ...     stopped=datetime(2025, 11, 10, 10, 30, 0),
   ...     run_by="engineer",
   ...     temp_dir="C:\temp",
   ...     project_dir="C:\project",
   ...     status="Normal Completion",
   ... )
   >>> batch.status
   'Normal Completion'
