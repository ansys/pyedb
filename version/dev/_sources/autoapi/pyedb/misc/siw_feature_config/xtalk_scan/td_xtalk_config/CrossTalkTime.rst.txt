   Add single ended net.


   :Parameters:

       **name** : :class:`python:str`
           Net name.

       **driver_rise_time** : :class:`python:float` or :class:`python:str`, :obj:`optional`
           Driver rise time value.
           The default is ``5.0``.

       **voltage** : :class:`python:float` or :class:`python:str`, :obj:`optional`
           Voltage value.
           The default is ``10``.

       **driver_impedance** : :class:`python:float` or :class:`python:str`, :obj:`optional`
           Driver impedance value.
           The default is ``5.0``.

       **termination_impedance** : :class:`python:float` or :class:`python:str`, :obj:`optional`
           Termination impedance value.
           The default is ``5.0``.



   :Returns:

       :ref:`bool <python:bltin-boolean-values>`
           ``True`` if the net was added successfully, ``False`` otherwise.










   .. rubric:: Examples

   >>> xtalk = CrossTalkTime(pedb)
   >>> xtalk.add_single_ended_net("DDR_DQ0", driver_rise_time=2.0, voltage=1.2)
