.. py:class:: pyedb.xml_parser.xml_stackup.XmlLayers(/, **data: Any)

   Container for layer definitions in the XML stackup.


   :Parameters:

       **length_unit** : :class:`python:str`, :obj:`optional`
           Unit for layer thickness measurements. The default is ``None``.

       **layer** : :class:`python:list` :obj:`of` :obj:`XmlLayer`, :obj:`optional`
           List of layer definitions. The default is an empty list.
