Examples
========
End-to-end examples show how you can use PyEDB. If PyEDB is installed on your machine,
you can download these examples as Python files or Jupyter notebooks and run them locally.

.. note::
    Some examples require additional Python packages.

AEDT integration
~~~~~~~~~~~~~~~~
The following examples illustrate the use of the legacy PyEDB API with PyAEDT.

.. note::
    In the examples, PyAEDT is configured not to display AEDT (`non-graphical=True`).
