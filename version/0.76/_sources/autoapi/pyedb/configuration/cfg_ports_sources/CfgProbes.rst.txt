   :Parameters:

       **name** : :class:`python:str`
           Unique probe name.

       **positive_terminal** : :class:`python:dict`
           Terminal-selector dictionary for the positive measurement node.

       **negative_terminal** : :class:`python:dict`
           Terminal-selector dictionary for the reference (ground) node.

       **reference_designator** : :class:`python:str`, :obj:`optional`
           Component reference designator.



   :Returns:

       :obj:`CfgProbe`
           The newly created probe object.
