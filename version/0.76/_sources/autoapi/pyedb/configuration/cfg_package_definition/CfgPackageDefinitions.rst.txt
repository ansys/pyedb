.. py:method:: add(name: str, component_definition: str, apply_to_all=None, components=None, maximum_power=None, thermal_conductivity=None, theta_jb=None, theta_jc=None, height=None, extent_bounding_box=None)

   
   Add a thermal package definition entry.


   :Parameters:

       **name** : :class:`python:str`
           Package definition name, e.g. ``"PKG_U1"``.

       **component_definition** : :class:`python:str`
           Component part-definition name.

       **apply_to_all** : :ref:`bool <python:bltin-boolean-values>`, :obj:`optional`
           Assign the package to every matching component when ``True``.

       **components** : :class:`python:list` :obj:`of` :class:`python:str`, :obj:`optional`
           Specific reference designators to target when *apply_to_all* is
           ``False``.

       **maximum_power** : :class:`python:str`, :obj:`optional`
           Maximum power dissipation, e.g. ``"5W"``.

       **thermal_conductivity** : :class:`python:str`, :obj:`optional`
           Package thermal conductivity.

       **theta_jb** : :class:`python:str`, :obj:`optional`
           Junction-to-board thermal resistance, e.g. ``"10C/W"``.

       **theta_jc** : :class:`python:str`, :obj:`optional`
           Junction-to-case thermal resistance, e.g. ``"5C/W"``.

       **height** : :class:`python:str`, :obj:`optional`
           Package height, e.g. ``"1mm"``.

       **extent_bounding_box** : :class:`python:dict`, :obj:`optional`
           Custom bounding-box extent override.



   :Returns:

       :obj:`CfgPackage`
           The newly created package object.  Call
           :meth:`CfgPackage.set_heatsink` on it to add fin geometry.










   .. rubric:: Examples

   >>> pkg = cfg.package_definitions.add(
   ...     "PKG_U1",
   ...     component_definition="IC_U1",
   ...     apply_to_all=True,
   ...     maximum_power="5W",
   ...     theta_jb="10C/W",
   ...     theta_jc="5C/W",
   ...     height="1mm",
   ... )
   >>> pkg.set_heatsink(fin_base_height="0.5mm", fin_height="3mm")
