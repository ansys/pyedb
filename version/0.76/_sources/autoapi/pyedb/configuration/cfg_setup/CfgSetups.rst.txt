.. py:method:: add_hfss_setup(config: CfgHFSSSetup = None, **kwargs)

   
   Create and register an HFSS setup.


   :Parameters:

       **config** : :obj:`CfgHFSSSetup`, :obj:`optional`
           Pre-built setup object, or *None* to construct a new setup from
           *kwargs*.

       **\*\*kwargs**
           Keyword arguments forwarded to :class:`CfgHFSSSetup` when *config*
           is *None*.  Common parameters:
           
           - ``name`` *(str, required)* — setup name.
           - ``adapt_type`` *(str)* — ``"single"`` | ``"broadband"`` |
             ``"multi_frequencies"``.  Default is ``"single"``.



   :Returns:

       :obj:`CfgHFSSSetup`
           The newly created (or registered) setup object.










   .. rubric:: Examples

   >>> hfss = cfg.setups.add_hfss_setup(name="hfss_1", adapt_type="broadband")
   >>> hfss.set_broadband_adaptive("1GHz", "20GHz")

.. py:method:: add_siwave_ac_setup(config: CfgSIwaveACSetup = None, name: str = None, si_slider_position: int = 1, pi_slider_position: int = 1, use_si_settings: bool = True, **kwargs)

   
   Create and register a SIwave AC setup.


   :Parameters:

       **config** : :obj:`CfgSIwaveACSetup`, :obj:`optional`
           Pre-built setup object, or *None* to construct from the remaining
           parameters.

       **name** : :class:`python:str`, :obj:`optional`
           Setup name.

       **si_slider_position** : :class:`python:int`, :obj:`optional`
           SI accuracy slider.  ``0`` = Speed, ``1`` = Balanced (default),
           ``2`` = Accuracy.

       **pi_slider_position** : :class:`python:int`, :obj:`optional`
           PI accuracy slider.  Same values as *si_slider_position*.

       **use_si_settings** : :ref:`bool <python:bltin-boolean-values>`, :obj:`optional`
           ``True`` (default) activates the SI slider rather than the PI
           slider.

       **\*\*kwargs**
           Additional keyword arguments forwarded to
           :class:`CfgSIwaveACSetup`.



   :Returns:

       :obj:`CfgSIwaveACSetup`
           The newly created (or registered) setup object.










   .. rubric:: Examples

   >>> siw = cfg.setups.add_siwave_ac_setup(name="siw_ac", si_slider_position=2)
   >>> siw.add_frequency_sweep("sw1", start="1kHz", stop="1GHz", step_or_count=100)

.. py:method:: get(name: str)

   
   Return an existing registered setup by name.

   Looks through the current :attr:`setups` list and returns the first
   entry whose ``name`` attribute matches *name*.  This is useful when
   you want to modify a setup that was registered earlier (e.g. by a
   template function) without having to keep a reference to the object.

   :Parameters:

       **name** : :class:`python:str`
           Setup name, e.g. ``"hfss_bb"`` or ``"siw_ac"``.



   :Returns:

       :obj:`CfgHFSSSetup` | :obj:`CfgSIwaveACSetup` | :obj:`CfgSIwaveDCSetup`
           The matching setup builder object.




   :Raises:

       :obj:`KeyError`
           If no registered setup with *name* exists.






   .. rubric:: Examples

   >>> cfg = edb.configuration.create_config_builder()
   >>> cfg.setups.add_hfss_setup("hfss_bb", adapt_type="broadband")
   >>> # ... later, retrieve and modify:
   >>> setup = cfg.setups.get("hfss_bb")
   >>> setup.add_frequency_sweep("sweep2", start="1GHz", stop="20GHz", step_or_count=100)
