.. py:method:: find_primitive(layer_name: str | list = None, name: str | list = None, net_name: str | list = None, prim_type: str | list = None, is_void: bool | None = None) -> list[pyedb.grpc.database.primitive.primitive.Primitive]

   
   Find primitive objects by one or more attributes.


   :Parameters:

       **layer_name** : :class:`python:str`, :class:`python:list`, :obj:`optional`
           Name of the layer or list of layer names.

       **name** : :class:`python:str`, :class:`python:list`, :obj:`optional`
           Name of the primitive or list of names.

       **net_name** : :class:`python:str`, :class:`python:list`, :obj:`optional`
           Net name or list of net names.

       **prim_type** : :class:`python:str`, :class:`python:list`, :obj:`optional`
           Primitive type filter, e.g. ``"polygon"`` or ``"path"``.
           Both lowercase (``"polygon"``) and EDB-style (``"Polygon"``) values are accepted.

       **is_void** : :ref:`bool <python:bltin-boolean-values>`, :obj:`optional`
           When ``True``, return only void primitives. When ``False``, return only non-void
           primitives. When ``None`` (default), void state is not used as a filter.



   :Returns:

       :class:`python:list`\[:obj:`Primitive`]
           Filtered list of primitives.
