.. py:method:: set_heatsink(fin_base_height=None, fin_height=None, fin_orientation=None, fin_spacing=None, fin_thickness=None)

   
   Attach heat sink fin geometry to this package definition.


   :Parameters:

       **fin_base_height** : :class:`python:str`, :obj:`optional`
           Base height of the fin array, e.g. ``"0.5mm"``.

       **fin_height** : :class:`python:str`, :obj:`optional`
           Fin height, e.g. ``"3mm"``.

       **fin_orientation** : :class:`python:str`, :obj:`optional`
           Fin orientation: ``"x_oriented"`` or ``"y_oriented"``.

       **fin_spacing** : :class:`python:str`, :obj:`optional`
           Center-to-center fin spacing, e.g. ``"1mm"``.

       **fin_thickness** : :class:`python:str`, :obj:`optional`
           Fin thickness, e.g. ``"0.2mm"``.



   :Returns:

       :obj:`CfgHeatSink`
           The newly created heat sink object.










   .. rubric:: Examples

   pkg.set_heatsink(
       fin_base_height="0.5mm",
       fin_height="3mm",
       fin_orientation="x_oriented",
       fin_spacing="1mm",
       fin_thickness="0.2mm"
   )
