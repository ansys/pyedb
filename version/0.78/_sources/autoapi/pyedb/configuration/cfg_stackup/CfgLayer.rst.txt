   Configure Huray surface roughness on selected surfaces.


   :Parameters:

       **nodule_radius** : :class:`python:str` , :class:`python:float`, :obj:`or` :class:`python:int`
           Huray nodule radius, e.g. ``"0.1um"``.

       **surface_ratio** : :class:`python:str` , :class:`python:float`, :obj:`or` :class:`python:int`
           Huray surface ratio.

       **enabled** : :ref:`bool <python:bltin-boolean-values>`, :obj:`optional`
           Enable roughness on this layer.  Default is ``True``.

       **top** : :ref:`bool <python:bltin-boolean-values>`, :obj:`optional`
           Apply roughness to the top surface.  Default is ``True``.

       **bottom** : :ref:`bool <python:bltin-boolean-values>`, :obj:`optional`
           Apply roughness to the bottom surface.  Default is ``True``.

       **side** : :ref:`bool <python:bltin-boolean-values>`, :obj:`optional`
           Apply roughness to the side surfaces.  Default is ``True``.



   :Returns:

       :obj:`CfgLayer`
           *self* — enables method chaining.










   .. rubric:: Examples
