





:class:`Siwave`
===============


.. py:class:: pyedb.siwave.Siwave(specified_version=None)

   Bases: :py:obj:`object`


   
   Initializes SIwave based on the inputs provided and manages SIwave release and closing.


   :Parameters:

       **specified_version** : :class:`python:str`, :class:`python:int`, :class:`python:float`, :obj:`optional`
           Version of AEDT to use. The default is ``None``, in which case
           the active setup is used or the latest installed version is used.
