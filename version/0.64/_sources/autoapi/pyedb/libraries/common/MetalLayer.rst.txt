





:class:`MetalLayer`
===================


.. py:class:: pyedb.libraries.common.MetalLayer(pedb, name, thickness=1e-06, material: str = 'Copper')

   Bases: :py:obj:`Layer`


   
   Convenience wrapper for metallic layers.

   Automatically creates a ``Conductor`` material.

   :Parameters:

       **pedb** : :obj:`ansys.edb.core.database.Database`
           Active EDB session.

       **name** : :class:`python:str`
           Layer name.

       **thickness** : :class:`python:float`, :obj:`optional`
           Thickness in meters.  Default is 1 Âµm.

       **material** : :class:`python:str`, :obj:`optional`
           Name of the conductor material.  Default is ``"Copper"``.











   .. rubric:: Examples

   >>> top = MetalLayer(edb, "TOP", thickness=18e-6, material="Gold")
   >>> top.material.conductivity
   58000000.0
