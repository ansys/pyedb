.. py:method:: add_net(name: str, is_bus: bool | str | int = False, is_clock: bool | str | int = False, is_critical: bool | str = False, net_type: str = 'Single-Ended', diff_mate_name: str = '') -> None

   
   Assign tags to a net.


   :Parameters:

       **name** : :class:`python:str`
           Name of the net.

       **is_bus** : :ref:`bool <python:bltin-boolean-values>`, :class:`python:str` or :class:`python:int`, :obj:`optional`
           Whether the net is a bus.
           The default is ``False``.

       **is_clock** : :ref:`bool <python:bltin-boolean-values>`, :class:`python:str` or :class:`python:int`, :obj:`optional`
           Whether the net is a clock.
           The default is ``False``.

       **is_critical** : :ref:`bool <python:bltin-boolean-values>` or :class:`python:str`, :obj:`optional`
           Whether the net is critical.
           The default is ``False``.

       **net_type** : :class:`python:str`, :obj:`optional`
           Type of the net.
           The default is ``"Single-Ended"``.

       **diff_mate_name** : :class:`python:str`, :obj:`optional`
           Differential mate name.
           The default is ``""``.













   .. rubric:: Examples

   >>> settings = EMCRuleCheckerSettings()
   >>> settings.add_net("CLK_100M", is_clock=True, is_critical=True)
   >>> settings.add_net("USB_DP", net_type="Differential", diff_mate_name="USB_DN")

.. py:method:: add_component(comp_name: str, comp_value: str, device_name: str, is_clock_driver: bool | str, is_high_speed: bool | str, is_ic: bool | str, is_oscillator: bool | str, x_loc: str | float, y_loc: str | float, cap_type: str | None = None) -> None

   
   Assign tags to a component.


   :Parameters:

       **comp_name** : :class:`python:str`
           Name of the component.

       **comp_value** : :class:`python:str`
           Value of the component.

       **device_name** : :class:`python:str`
           Name of the device.

       **is_clock_driver** : :ref:`bool <python:bltin-boolean-values>` or :class:`python:str`
           Whether the component is a clock driver.

       **is_high_speed** : :ref:`bool <python:bltin-boolean-values>` or :class:`python:str`
           Whether the component is high speed.

       **is_ic** : :ref:`bool <python:bltin-boolean-values>` or :class:`python:str`
           Whether the component is an IC.

       **is_oscillator** : :ref:`bool <python:bltin-boolean-values>` or :class:`python:str`
           Whether the component is an oscillator.

       **x_loc** : :class:`python:str` or :class:`python:float`
           X coordinate.

       **y_loc** : :class:`python:str` or :class:`python:float`
           Y coordinate.

       **cap_type** : :class:`python:str` or :data:`python:None`, :obj:`optional`
           Type of the capacitor. Options are ``"Decoupling"`` and ``"Stitching"``.
           The default is ``None``.













   .. rubric:: Examples

   >>> settings = EMCRuleCheckerSettings()

   ...     comp_name="U1",
   ...     comp_value="FPGA",
   ...     device_name="XC7A35T",
   ...     is_clock_driver=False,
   ...     is_high_speed=True,
   ...     is_ic=True,
   ...     is_oscillator=False,
   ...     x_loc="10.5",
   ...     y_loc="20.3",
